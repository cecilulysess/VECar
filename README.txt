VECar v. 0.9 README:

To build and execute the program:
1. Open the TestingCourse scene in Unity3D.
2. Go to File -> Build Settings, target a Windows machine, and click Build.
3. This will allow you to save the executable to wherever you please.
4. Open the executable and choose your resolution and graphics settings as well as any changes to the input you would like.
5. The important input fields are vertical (acceleration), horizontal (steering), and reverse.
6. Play!
